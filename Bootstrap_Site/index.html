About us

TomgaWeb tool

The TomgaWeb tool allows you to easily create a website on your own. No programming skills are needed to use it, as you can create a website simply by draging components like titles, paragraphs, buttons and many more on the screen. It even has a built in databse and a fully responsive layout.

TomgaApp

After creating a website, you can use our TomgaApp to see it's activity and popularity. You'll be abel to see stats such as, visits, interactions, searches and even your databases data. The TomgaWeb tool is also available in the app, so you can create and update your website on the go.

Want a more specific website? Contat us, and we will create your dream website for you.

Getting started

To start creating your own website, simply open the TomgaWeb tool or download the TomgaApp. We suggest starting with the web tool, as it has more features. Once you have opened the web tool, you can immediately start creating your website, but we would suggest watching this short guid on how to use it. Good luck!


Once you have created a website, open the TomgaApp and link it with the website. You will need to confirm you email to do this. After that you can start tracking and updating your website from anywhere you want.
